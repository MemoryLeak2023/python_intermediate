## Voorbeeldcode:

```python
from abc import ABC, abstractmethod

class Animal(ABC):
    def __init__(self, name):
        self.name = name

    @abstractmethod
    def make_noise(self):
        pass

class Cat(Animal):
    def make_noise(self):
        print(f"{self.name} says, Meow!")

class Dog(Animal):
    def make_noise(self):
        print(f"{self.name} says, Woof!")

kitty = Cat("Maisy")
doggy = Dog("Amber")

kitty.make_noise()  # Output: "Maisy says, Meow!"
doggy.make_noise()  # Output: "Amber says, Woof!"
```
